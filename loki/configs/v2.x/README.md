# Loki 2.x Configurations

Stable Loki 2.x monolithic configurations for development.

## Available Configurations

- `loki-2.9.14-loki-config-prod.yaml` - Production-ready configuration
- `loki-2.9.14-s3-memberlist-config.yaml` - S3 storage with memberlist

## Features

- Proven stability
- Extensive documentation
- Production battle-tested
- Backward compatibility

## Usage

```bash
# Start Loki with specific config
loki -config.file=loki-2.9.14-s3-memberlist-config.yaml
```
